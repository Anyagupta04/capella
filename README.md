# Capella 🌟
### Reimagining Transit Safety — "Uber for Women, by Women"

Capella is a high-fidelity, interactive concept web platform designed to address critical safety gaps in modern ride-hailing services. Built exclusively using **semantic HTML5, custom responsive CSS3, and lightweight Vanilla JavaScript**, it is completely self-contained, client-side, and ready for instant deployment.

## 🚀 Key Features

### 1. Passenger Safety Center
*   **Live Route Tracking**: Simulated real-time coordinate syncing along a winding SVG street grid route. Calculates trip ETA based on path progress.
*   **Encrypted Connections**: Displays active Route Guard status alerts monitoring off-route deviations.
*   **Interactive SOS Emergency System**: An immediate response warning dashboard triggering a pulsing visual alarm and a 5-second countdown to alert police nodes.
*   **Smart Buddy Companion Chat**: A dedicated safety buddy dialogue box enabling direct messaging with instant, context-aware checking responses.

### 2. Driver Partner Hub
*   **Fair Compensation Model**: Explains Capella's developer revenue model paying drivers a 90% fare split (pioneering over the industry's traditional 70% average).
*   **Earnings Calculator Slider**: Allows prospective partners to adjust hours worked, average rides, and fare prices to estimate weekly, monthly, and annual net payouts instantly.
*   **Registration Application**: A secure partner portal form allowing drivers to apply with credentials.

### 3. Business Strategy & Expansion
*   **Scalable Monetization**: Clear overview of four revenue channels (individual commissions, pooled rides, subscription tiers, and corporate B2B contracts).
*   **Expansion Timeline**: System roadmap tracking scaling phases from major Metro hubs to regional Tier-2 cities and multi-modal integrations.
*   **Data Projections Graph**: Animated custom CSS charts visualizing Gross Booking Value (GBV) target scales.

## 🛠️ Technology Stack
*   **Front-End Structure**: HTML5 (Semantic elements)
*   **Styling**: CSS3 (Flexbox, CSS Grids, custom property design systems, keyframe animations, glassmorphic UI cards)
*   **Simulated Logic**: Vanilla JavaScript (ES6+, DOM event listeners, SVG path length offset animations, mathematical calculations)
*   **Build Tooling**: None (Zero frameworks, dependencies, npm configurations, or transpilers required—pure static files).

## 💻 Local Setup & Development
Since the project is built with static code assets, you can run it locally in under 5 seconds:
1. Clone this repository:
   ```bash
   git clone https://github.com/your-username/capella.git
   ```
2. Open the directory:
   ```bash
   cd capella
   ```
3. Open `index.html` in your favorite web browser (double-click the file, or right-click &rarr; "Open with Google Chrome / Microsoft Edge").
